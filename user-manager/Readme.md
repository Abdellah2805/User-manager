# Projet User-Manager

## Descriptions

Ce projet est une application web complète (full-stack) conçue pour la gestion des utilisateurs. Elle est composée d'un système d'authentification robuste (inscription et connexion) et d'un tableau de bord d'administration qui permet de visualiser et de gérer une liste d'utilisateurs.

L'objectif principal est de montrer l'interaction entre un **frontend** (HTML/CSS/JavaScript) et un **backend** (Node.js/Express) via une API REST.

## Fonctionnalités

* **Système d'authentification** : Les utilisateurs peuvent s'inscrire et se connecter en toute sécurité.
* **Tableau de bord sécurisé** : L'accès au tableau de bord est protégé par un token d'authentification.
* **Gestion des utilisateurs** : Le tableau affiche une liste d'utilisateurs. La suppression d'utilisateurs est simulée directement depuis l'interface.
* **Interface utilisateur professionnelle** : Le design est moderne, sombre et responsive, offrant une expérience utilisateur agréable.
* **API externe** : Le projet utilise une API publique (JSONPlaceholder) pour récupérer et afficher une liste de données d'utilisateurs, ce qui simule l'utilisation d'une base de données.

## Technologies utilisées

* **Frontend** :
    * **HTML5**
    * **CSS3**
    * **JavaScript (ES6+)**
    * **Font Awesome** pour les icônes
* **Backend** :
    * **Node.js** (environnement d'exécution)
    * **Express.js** (framework web)
